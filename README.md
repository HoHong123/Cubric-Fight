  <h1>요약</h1>
  <div>
    <p>
      한줄 요약을 하면 알까기 게임을 생각하시면 될 것 같습니다.
      상대를 밀어 맵 밖으로 보내는 기본 알까기와는 달리 캐릭터를 밀어서 나오는 속도에 비례하여 데미지를 주고 적의 체력이 0이
      되면 죽게하여 모든 적을 처리하면 이기는 게임입니다. 작성자는 이 프로젝트에 "보스대전" 컨텐츠의 보스의 AI에 유한행동기계와
      맵 디자인 및 레벨 디자인을 제작 및 관리하였습니다.
    </p>
  </div>
